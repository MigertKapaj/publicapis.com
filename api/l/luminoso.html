This API supports the following methods. Unless otherwise specified, all of them return a JSON object with two keys, "error" and "result". If there is no error, "error" will be null and "result" will be what the "Result" lines below indicate. If there is an error, "result" will be null; "error" will be a JSON object with "code" (a basic error code) and "message" (a human-readable error message).
