Postmark’s spam

Postmark’s spam API is a RESTfull interface to the Spam filter tool SpamAssassin.This can be used to pre or post process email content when it is used in your application. For instance, if you process incoming emails from your users you can first run it through the API to filter out junk and avoid clutter in your database and application. Keep in mind: This is free to use and may be updated, removed or changed at any time.
