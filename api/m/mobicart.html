MobiCart

The MobiCart API follows the REST principles and has been built as vanilla XML/JSON over HTTP using GET/POST/PUT/DELETE. Every resource such as Post, Comment or Task has its own URL and can be manipulated separately.You can even explore the view part of the API (everything that's fetched with GET) through a regular browser. Using Firefox for this is preferable, as it has a good, simple XML renderer (unlike Safari which just strips the tags and dumps the content). Almost all URLs in MobiCart can be viewed in their XML form by adding the .xml extension.
