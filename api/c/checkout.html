Checkout

The Checkout application programming interface (API) is a protocol from Paytech that allows you to post shopping carts from your web site or your Interactive Voice Response module to your payment platform. The API also enables you to charge credit cards, debit bank accounts (ACH) and refund buyers. An API integration is a good choice for merchants who want to automate their order processing and online payments.
