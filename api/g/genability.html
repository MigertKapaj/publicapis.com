Genability

Echo is a simple call that allows you to validate how you are calling the Genability APIs. This is designed for the user who is just beginning to use our APIs as well as for debugging help. For example, you can call the /validatedate method to verify the date format you are passing in with your requests is valid. As with all calls, you will need to use a valid app ID and key to make these calls.
