The Ringio CRM

The Ringio CRM API is a RESTful API that allows you to read and write Ringio data, such as contacts, notes, phone calls and users. With the API you can develop integrations with CRM systems, such that when Ringio users get phone calls, the callers can be identified from information stored in a CRM system. You can also enrich the CRM with data captured by Ringio, such as logging phone calls as interaction records in the CRM.
