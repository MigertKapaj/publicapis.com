Thomson Reuters Tick History

RDTH API is a request-retrieve web service that provides access to a wide range of Thomson Reuters Tick History content. The TRTHApi Data Web Services consist of operations for build data dictionary constructs, searching for instruments, and requesting intra-day market data. This web service is both language and platform independent, allowing flexibility and easy integration of Reuters market data into your own internal systems.
