TideSpy Raw Data

The TideSpy Raw Data API calls accept web (HTTP) requests and return JSON-formatted responses. All requests must contain a valid key which is assigned when signing up for this service. The following request types are supported: The places (locations) for which the other data is availablePlace name searchTide height data for a range of datesWind and Swell forecasts, both in numeric terms and human-friendly text, for the next 6 daysSun and Moon rise and set times and the phase of the moon, for a given range of dates
