The vCloud Express

The vCloud Express API provides secure RESTful access to many of the features and functions available within the My Cloud User Interface. The RESTful API is easy to understand, implement, and compatible with most programming and scripting languages. Since the vCloud Express API extends your control over the My Cloud User Interface, we’ve included additional security to protect your account against unauthorized access Access to the RESTful API is secured via SSL and your My Cloud user credentials.
