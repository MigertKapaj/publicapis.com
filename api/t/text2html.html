Text2HTML

Converts text to HTML Start a line with h1, h2 or h3 to create a header. We will convert text that looks like URLs to hyperlinks to that URL http://foo.com => <a href='http://foo.com'>http://foo.com</a> We try to leave valid HTML alone. For example, if you include <p>http://foo.com.</p> then we will not add an anchor tag to that URL. Non-HTML friendly characters like & < are escaped like & < We do minimal conversion of text. This approach differs (by design) from http://txt2html.sourceforge.net/ which does more processing like numbered lists and headers and tables. Feedback on this tool is welcomed on our blog.
